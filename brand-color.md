- color.black = #000000
- color.white = #FFFFFF

### Secondary Colors
- color.lightGrey = #D8D6D0
- color.sage = #79897A
- color.darkOlive = #585747
- color.blueGrey = #B3C1C7

---

## Semantic Usage

### Text
- text.primary = color.black
- text.secondary = color.primaryGrey
- text.inverse = color.white

### Backgrounds
- bg.primary = color.white
- bg.secondary = color.cream
- bg.dark = color.black

### Accents
- accent.primary = color.sage
- accent.secondary = color.blueGrey

---

## Rules

- NEVER introduce new colors
- Avoid saturated / bright colors
- Use primary colors for 80% of UI
- Use secondary colors only as accents
